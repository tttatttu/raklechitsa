### Инструкция по запуску проекта:

```
  # установка зависимостей
  $ npm install

  # запуск сервера с "горячей" перезагрузкой на localhost:3000
  $ npm run dev

  # сборка в продакшн и запуск сервера
  $ npm run build
  $ npm run start

  # генерация статического проекта
  $ npm run generate
```
